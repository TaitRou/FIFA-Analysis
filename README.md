# FIFA-Analysis
Find the needle in a haystack with my excel based query system on 18,000+ players

I personally am a huge fan of soccer and have played FIFA video games my whole life so I figured
why not build a product I could actually use. By filtering the raw database by custom parameters,
not only can I identify potential signings, but I can also get a great deal on them! Try out the 
search yourself by using the 4 highlighted fields and the players matching set criteria will populate
